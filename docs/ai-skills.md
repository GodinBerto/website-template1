## Icon Usage (Lucide)

- Use Lucide React for all icons in the UI
- Import icons from "lucide-react"
- Icons must be used as React components

Tools:

- lucide-react

Rules:

- Do NOT use any other icon libraries (e.g., FontAwesome, Heroicons)
- Only import the icons that are needed (no bulk imports)
- Keep icon size consistent (default: 16px, 20px, or 24px)
- Use Tailwind classes for styling (e.g., text-gray-500, w-5 h-5)
- Icons must align properly with text and components
- Use semantic icons (e.g., ShoppingCart for cart, User for profile)

Output:

- Clean icon imports and usage inside components
